# Gnuplot Scripts for Physics Animations

このディレクトリには、物理現象を視覚化するための Gnuplot スクリプト（`.gp` ファイル）と、それによって生成されたアニメーション GIF が含まれています。

## 概要

各 `.gp` ファイルは、物理法則に基づいた数式を使用して、現象の変化を時系列でプロットし、アニメーション GIF を作成するように設計されています。

## ファイルの詳細

| ファイル名 | 現象 | 内容 |
| :--- | :--- | :--- |
| `beats.gp` | うなり | わずかに異なる周波数の2つの波の重ね合わせと包絡線を描画します。 |
| `doppler_effect.gp` | ドップラー効果 | 移動する波源から放出される球面波（波面）をアニメーション化します。 |
| `huygens_principle.gp` | ホイヘンスの原理 | 素元波の重ね合わせによって新しい波面が形成される様子を示します。 |
| `longitudinal_wave.gp` | 縦波 | メディアの粒子の疎密変化をアニメーションで表現します。 |
| `refraction_snell.gp` | 屈折 | スネルの法則に基づいた、境界における波の進行方向の変化を描画します。 |
| `standing_wave_*.gp` | 定常波 | 弦や気柱（開管・閉管）における定常波の形成と倍振動を可視化します。 |
| `wave_interference_*.gp` | 波の干渉 | 2つの波源からの波が干渉し、強め合う点と弱め合う点ができる様子を3Dや俯瞰図で示します。 |

## 生成ツールの使用方法

- `generate_gifs.py`: Python を使用して複数のスクリプトから一括で GIF を生成します。
- `run_all.bat`: Windows バッチファイルを使用して、環境内の Gnuplot を呼び出し一括実行します。
- `RemoveBOM.cs`: ファイルから BOM（Byte Order Mark）を削除するための C# スクリプトです。

## 必要条件

これらのスクリプトを実行するには、[Gnuplot](http://www.gnuplot.info/) がインストールされている必要があります。
